Set Timeout

Time interval after which the calculator will produce an error if no result has been found for a given execution.

Certain operations (fractional and decimal exponents, in particular) may take longer than a few seconds to complete. Specity the maximum amount of time to wait for a result before throwing an error.

Rounding

Specify the maximum number of decimal places you would like to see in your results.
