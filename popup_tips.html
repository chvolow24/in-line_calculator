+ - * / ^ ( )

In most text areas on most web pages:

1. Type [[ to "open" the calculator.

2. Type the expression you want to evaluate

3a. Type == to evaluate and leave the calculator "open", OR

3b. Type ]] to "close" the calculator.

[[2+2==
 -->    [[4

[[4 -(4*.2) ==
 -->    [[3.2

[[2.2 - 5 ]]
 -->    -3.2

[[(.5*.44)^(1/5)+12]]
 -->    12.7387

Format results

Type ,= to format the result with commas, or $= to format the result as a currency.

[[2,243 ^ 2 ,=
 -->    [[5,031,049

[[2459.33 * 1.02^50 $=
 -->    [[$6,619.50
